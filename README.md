# Twentig Blueprints

WordPress Playground blueprints for previewing and testing WordPress sites.

## What are Blueprints?

Blueprints are WordPress setup scripts that instantly launch a pre-configured WordPress site — no installation needed. They define themes, plugins, content, and settings in a single JSON file.

## Open a Blueprint in:

- [Twentig Playground](https://twentig.com/playground)
- [WordPress Playground](https://playground.wordpress.net)
- [WordPress Studio](https://developer.wordpress.com/studio/)
